🪝 HoloClaw

Multiplayer AI workspace on Hologram. OpenClaw is single-player — HoloClaw is a team.

What makes it different:
  • Multiple verified users share one live AI workspace, each on their own encrypted DIDComm channel.
  • Role-based tool access: owner, collaborator, observer, approver.
  • Runtime tool curation — admins paste MCP server URLs from the chat; the agent gets new tools instantly.
  • Live tool execution feed — every member sees what the agent is doing in real time.
  • Approval gates — sensitive tool calls require sign-off from a designated approver.
  1. Install the Hologram app from the App Store or Play Store.
  2. Open the app and tap the scan / connect button.
  3. Scan the QR code above — you'll land in the lobby.
  4. Tap Create workspace to start a new one, or paste an invite token to join.
  5. Once in a workspace, tap Invite a member to get a token to share with a teammate.
